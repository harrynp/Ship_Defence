Spaceship Defence version 1.0
=============================
Run from ShipDefence.py.

Controls
========
Up = Moves Red Paddle Up
Down = Moves Red Paddle Down
W = Moves Blue Paddle Up
S = Moves Blue Paddle Down
P = Pauses Game And Unpauses Game If Paused

Bugs
====
1. Only way out of credits and control screen are pressing arrow keys to show the menu.
2. Can't use both paddles at the same time. The WASD overrides the arrow keys and vice versa.
3. Walls do not detect the collision of the ball when the ball intially goes through 
the paddle (usually from behind).
